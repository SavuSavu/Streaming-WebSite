Design Details

  • All pages contain a navigation bar that is located at the top of the page, and is using a classic color scheme with a black background and the white text for better readability

  • As is stated in WCAG 2.0 every not-textual content has a text alternative

  • All pages can be accessed using the keyboard and the user can see all the content as is stated in WCAG 2.0

  • According to Jennifer K.(2018) for better readability of tables in general the best way to go is to use Zebra Striped Tables design because every even/odd row has a different color and this helps the user to focus on the content

  • For a better responsiveness design all of the text and images in this web application are scalable according to the size of the page

  • Research has shown that 3d graphics or depth cues can be used in web design to improve the users' visual attention (Anderson,2017)
